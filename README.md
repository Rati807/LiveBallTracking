# Smart Cricket Ball Tracker AI

## Description
This project detects a cricket ball in real time using AI (OpenCV) and shows its position, direction, and status on a web interface.

## Features
- Ball Detection
- FOUND / LOST Status
- Direction (LEFT / RIGHT)
- Distance (NEAR / FAR)

## Technology Used
- Python
- OpenCV
- Flask
